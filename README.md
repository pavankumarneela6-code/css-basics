# Explore India Tourism Website

## 📖 Project Description
This is a tourism website created using HTML and CSS. It showcases popular tourist destinations, foods, transportation options, tour packages, travel statistics, adventure activities, booking forms, galleries, and customer reviews.

## 🚀 Features
- Navigation Menu
- Popular Destinations Section
- Popular Foods Section
- Transportation Information
- Tour Packages Table
- Travel Statistics
- Adventure Activities
- Packing Checklist
- Travel Survey
- Budget Planner
- Image Gallery
- Booking Forms
- Customer Reviews
- Travel Videos
- Contact Information

## 🛠️ Technologies Used
- HTML5
- CSS3
